1. En un archivo de texto separado que debes crear, escribe explicaciones de los siguientes conceptos como si se lo estuvieras explicando a un niño de 12 años. Hacer esto te ayudará a descubrir rápidamente cualquier agujero en tu comprensión.

	* Variables
	* Strings
	* Funciones (argumentos, `return`)
	* Declaraciones `if`
	* Valores booleanos (`true`, `false`)

Variables = espacio de memoria reservado para un valor que sera utilizado luego. Debe ser definido y nombrado. Pueden pertenecer a varios tipos y ser o no editables luego de su declaracion

Strings = variables de texto formadas por caracteres concatenados

If = declaracion condicional. Se establece una condicion si se cumple ejecuta una porcion de codigo y si no se cumple puede ejecutar otra utilizando else

Valores booleanos = tipo de valor que solo puede adquirir uno de dos estados posibles, true (1) o false (0)
